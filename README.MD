LABYRINTHE ALEATOIRE

Ce programme affiche un labyrinthe aléatoire

F1 : Lance le creuseur qui va creuser aléatoirement le labyrinthe
F2 : démarre une partie avec un joueur contrôlé par les flèches du clavier en haut à gauche
     et deux fantômes (un qui pacourt main gauche, l'autre qui parcourt main droite)
F3/ F4 : ajoute un fantôme supplémentaire, de couleur aléatoire
